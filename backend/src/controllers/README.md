# Carpeta `controllers`

En esta carpeta se ubicarán los controladores de la aplicación, encargados de manejar la lógica de las rutas y procesar las solicitudes HTTP.

Por ejemplo: un controlador puede recibir una solicitud para crear un usuario, validar los datos recibidos, llamar al servicio correspondiente para crear el usuario en la base de datos y devolver una respuesta al cliente.
