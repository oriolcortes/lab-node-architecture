# Carpeta `services`

En esta carpeta se almacenarán los servicios de la aplicación, donde se implementa la lógica de negocio y las operaciones que utilizan los controladores. Los servicios suelen utilizar los repositorios para acceder y manipular los datos en la base de datos.

Por ejemplo: un servicio puede validar si un usuario tiene permisos antes de permitirle actualizar un recurso, o calcular el total de una orden antes de guardarla en la base de datos.
