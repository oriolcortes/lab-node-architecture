# Carpeta `validators`

En esta carpeta se almacenarán los validadores, que se encargan de comprobar que los datos recibidos en las peticiones cumplen con los requisitos necesarios (tipo de dato, formato, campos obligatorios, etc.).

Por ejemplo: un validador puede comprobar que un email tenga el formato correcto o que una contraseña tenga una longitud mínima.
